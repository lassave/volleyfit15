VolleyFit 15 - PWA listo para GitHub Pages
===========================================

Contenido:
- index.html
- styles.css
- manifest.json
- service-worker.js
- icons/

Pasos para publicar en GitHub Pages:
1. Crear un repositorio en GitHub (público o privado).
2. Subir todos los archivos de esta carpeta al repo (root).
   Puedes hacerlo por Git o subiendo el ZIP descomprimido en la web de GitHub.
3. En GitHub: Settings -> Pages -> Branch: select 'main' (o 'gh-pages') and folder: '/ (root)'. Save.
4. Esperar unos minutos. La app estará disponible en:
   https://TU_USUARIO.github.io/NOMBRE_REPO/

Notas iPhone:
- Abrir la URL en Safari, tocar el botón compartir -> "Añadir a pantalla de inicio".
- Para que funcione como PWA, GitHub Pages debe servirlo por HTTPS (lo hace por defecto).
